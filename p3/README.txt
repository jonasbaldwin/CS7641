# Instructions for Project Three

* download and install Anaconda (including Jupyter)
* download the code from the Git repository located at https://github.com/mahonbaldwin/CS7641
* from the `/p3` directory
    * install the requirements (from the `requirement.txt` file).
    * run `jupyter notebook`
    * open http://localhost:8888/tree from a browser (if you're using the default configuration)
    * open each of the notebooks (uber-tests and scatter-plots).
    * run the notebooks using the buttons at the top of the page.
